‚ùì Frequently Asked Questions

Why do user IDs start at 43, 44, 45 instead of 1, 2, 3?

This is completely normal. The IDs in the central database represent actual registration order across all users. IDs 1-42 were earlier registrations. Your phone number is the unique identifier, not the ID. Local emergency records always start from 1 on each device.

Why is internet required for registration?

Security feature - prevents duplicate accounts and ensures central validation. After one-time registration, app works 100% offline.

What happens if I uninstall?

All local data is permanently deleted - emergency records, off days, statistics. This is Android's standard behavior. Always export PDFs regularly!

Can I use on multiple devices?

Yes, but each device maintains independent emergency records. Your profile (name, phone) syncs with central database.
